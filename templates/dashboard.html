📡 API Endpoints

GET /api/status → Stato sistema per AI Agent (con lock bidirezionale)
GET /api/dashboard/status → Stato completo per Dashboard UI
GET /api/status/esterno → Contesto esterno minimo (manuale/esterno, non usato dalla dashboard)
GET /api/status/{stanza} → Stato stanza + contesto esterno
GET /health → Health check servizio (monitoring esterno)
POST /api/{stanza}/{attuatore} → Controlla dispositivo
POST /api/presence → Segnala presenza/assenza e notifica n8n
POST /api/actions/scenario/{nome} → Scenario (manuale/API esterna)
POST /api/query → Query mirate (manuale/API esterna)
POST /api/time → Imposta ora simulata (manuale/API esterna)
POST /api/time-speed → Velocità tempo (0-600 min/min)
POST /api/day → Imposta giorno della settimana
POST /api/season → Imposta stagione
POST /api/weather → Imposta condizione meteo
POST /api/n8n-interval → Intervallo AI Agent (10-300s)
POST /api/pause → Pausa/riprendi simulazione
POST /api/debug → Attiva/disattiva modalità debug
GET /api/preferences → Tutte le preferenze utente
GET /api/preferences/{stanza} → Preferenze stanza specifica
PUT /api/preferences/{stanza} → Modifica preferenze stanza
DEL /api/preferences/{stanza} → Reset preferenze stanza
DEL /api/preferences → Reset preferenze globali (tutte le stanze)
POST /api/agent/action → Log azione AI Agent (uso agent/integratori esterni)
GET /api/logs/agent → Log agenti (con filtri)
DEL /api/logs/agent → Cancella log agenti
DEL /api/logs/agent/delete?agent=...&date=... → Cancella log filtrati per agente e/o data
GET /api/logs/analysis → Analisi e statistiche log
